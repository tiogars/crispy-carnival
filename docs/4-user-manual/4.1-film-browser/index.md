# 4.1 Film Browser

## Steps

1. Start the FastAPI server from the `server` folder.
2. Start the Vite frontend from the project root.
3. Open the webapp in your browser.
4. Select a film.
5. Select a reel sequence.
6. Use player controls to inspect frames.

## Notes

- If no films are shown, verify `FILM_LIBRARY_ROOT` and folder structure.
- If no frames are shown, verify image extensions and naming.
